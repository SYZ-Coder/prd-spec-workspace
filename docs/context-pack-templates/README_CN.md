# 标准上下文包模板

这个目录提供三类“可直接复制”的上下文模板，帮助团队把当前需求产物快速整理成可喂给不同目标工具的输入包。

适用目标：

- OpenSpec
- Superpowers
- 通用 AI / 开发 AI

使用方式：

1. 打开对应模板
2. 按模板中的文件清单复制当前需求的实际路径
3. 删除不适用项
4. 将 `<change-name>`、`<domain>`、`<title>` 等占位符替换为真实值
5. 把整理后的上下文整体复制给目标工具
